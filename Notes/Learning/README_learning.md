# Learning Notes

Ce dossier contient des notes pedagogiques pour comprendre les concepts techniques utilises dans le projet Unity.

## Index

- `Event_Listener_Unity_CSharp.md` : comprendre les evenements (`event`) et les listeners en C# / Unity, avec exemples concrets.

## Comment utiliser ces notes

1. Lire le concept en haut de fiche.
2. Reproduire l'exemple minimal dans un projet test.
3. Adapter ensuite au projet principal (farm, inventaire, UI).
4. Ajouter en bas de fiche vos retours ("ce que j'ai compris", "ce qui bloque").
